Dear diary,

Bandung, December 16th 2020
      Waktu saya berumur sekitar 5 tahun, saya sempat memiliki cita-cita sebagai seorang dokter.
      Namun setelah saya tahu bahwa teman saya yang memiliki ibu seorang dokter, saya menjadi tidak menginginkan profesi itu lagi.
      Kehidupan sosial temen saya menjadi kurang baik. Dia memiliki ibu yang sibuk dan saya tidak menginginkan hal yang sama terjadi pada anak saya nanti.
      Saya senang jika mommy saya selalu berda disamping saya. Dan menurut saya semua anak memiliki pendapat yang sama.
  
      Di waktu saya duduk di bangku SD, saya bingung ingin menjadi apa dimasa depan.
      Dan suatu hari saya menang untuk lomba CALISTUNG (Membaca, Menulis dan Berhitung). Saya menceritakannya kepada mommmy saya.
      Beliau tidak percaya dan menganggap mungkin guru yang disekolah salah membaca nama pemenang. Disitu saya benar-benar patah semangat.
      Seakan-akan saya tidak sepandai yang dipikirkan. Tapi akhirnya, pengumuman pemenang keluar dan saya mendapatkan piala kejuaraan saya.
      Saya senang dapat membuktikannya jika memang saya lah yang menang dalam lomba CALISTUNG.
      Saya menatap matanya, namun tidak ada reaksi bangga.... Hal itu mungkin masih saya ingat namun tidak apa-apa lagi sekarang.
  
      Dimasa yang sama, saya juga suka menggambar dan mewarnai.
      Sayangnya itu tidak pernah dilihat sebagai suatu anugrah.
      Saya sering dimarahi waktu kecil karena diki-dikit menggambar dan mewarnai.
      Suatu kali pernah saya membuat karya. Tapi sayangnya hasil gambaran saya dibuang pada saat saya sedang tidur siang.
      Saya pikir ada benarnya, saya dimarahi karena memang menggambar dan mewarnai banyak membuang waktu saya, yang dimana itu dapat digunakan untuk belajar.
      Tapi karena saya suka sekali menggambar dan mewarnai, saya curi-curi untuk bisa melalukkannya.
  
      Hingga sampainya saya duduk di bangku SMA. Pada saat itu, belum pilih jurusan IPA atau mau IPS.
      Dan setelah naik ke kelas 2 SMA saya disuruh unutk mengabil kelas IPA dan mengikuti tes. 
      Saya lulus masuk ke kelas IPA sedangkan saya menyukai pelajaran Geografi.
      Waktu itu saya belum memahami betapa pentingnya dan bergunanya mengambi kelas IPA.



      to be continuet
